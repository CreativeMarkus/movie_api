Movie API Documentation

This API allows users to get information about movies, genres, directors, and manage user accounts and favorites.

GET /movies

Returns a list of all movies.

Request:
GET /movies

Response:
[
  {
    "id": "tt0111161",
    "title": "The Shawshank Redemption",
    "director": "Frank Darabont",
    "genre": "Drama",
    "releaseYear": 1994
  },
  ...
]

GET /movies/:id

Returns information about a specific movie by ID.

Request:
GET /movies/tt0111161

Response:
{
  "id": "tt0111161",
  "title": "The Shawshank Redemption",
  "director": "Frank Darabont",
  "genre": "Drama",
  "releaseYear": 1994
}

GET /genres/:name

Returns information about a specific genre.

Request:
GET /genres/Drama

Response:
{
  "name": "Drama",
  "description": "Emotionally driven and serious"
}

GET /directors/:name

Returns information about a specific director.

Request:
GET /directors/Christopher Nolan

Response:
{
  "name": "Christopher Nolan",
  "bio": "Known for Inception, Dark Knight",
  "birthYear": 1970
}

POST /users

Registers a new user.

Request:
POST /users

Body:
{
  "username": "john123",
  "password": "securepass",
  "email": "john@example.com"
}

Response:
{
  "message": "User created.",
  "user": {
    "username": "john123",
    "email": "john@example.com"
  }
}

POST /login

Authenticates a user and returns a token.

Request:
POST /login

Body:
{
  "username": "john123",
  "password": "securepass"
}

Response:
{
  "message": "Login successful.",
  "token": "JWT-TOKEN-HERE"
}

PUT /users/:username

Updates user information.

Request:
PUT /users/john123

Body:
{
  "email": "newemail@example.com",
  "password": "newsecurepass"
}

Response:
{
  "message": "User updated.",
  "user": {
    "username": "john123",
    "email": "newemail@example.com"
  }
}

POST /users/:username/favorites/:movieId

Adds a movie to the user's list of favorites.

Request:
POST /users/john123/favorites/tt0111161

Response:
{
  "message": "Movie added to favorites.",
  "user": {
    "username": "john123",
    "favorites": ["tt0111161"]
  }
}

DELETE /users/:username/favorites/:movieId

Removes a movie from the user's list of favorites.

Request:
DELETE /users/john123/favorites/tt0111161

Response:
{
  "message": "Movie removed from favorites.",
  "user": {
    "username": "john123",
    "favorites": []
  }
}

DELETE /users/:username

Deletes the user account.

Request:
DELETE /users/john123

Response:
{
  "message": "User deleted."
}

For questions or updates, please contact the API developer.
