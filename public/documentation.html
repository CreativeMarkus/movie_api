MyFlix API Documentation

This API allows users to access information about movies and manage user accounts. It supports basic CRUD operations for users and allows movie data retrieval.

Endpoints

GET /movies

Returns a list of all movies.

  • Method: GET
  • URL: /movies
  • Query Parameters: None
  • Request Body: None
  • Response:
    • [
  {
    "title": "Inception",
    "director": "Christopher Nolan"
  },
  {
    "title": "The Matrix",
    "director": "The Wachowskis"
  }
]
GET /movies/:title

Returns a single movie by its title.

  • Method: GET
  • URL: /movies/Inception
  • Query Parameters: None
  • Request Body: None
  • Response:
    • {
  "title": "Inception",
  "director": "Christopher Nolan"
}
POST /users

Adds a new user to the database.

  • Method: POST
  • URL: /users
  • Query Parameters: None
  • Request Body:
    • {
  "name": "Your Name",
  "email": "email@example.com"
}
  • Response:
    • {
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "name": "Your Name" ,
  "email": "email@example.com"
}
PUT /users/:id

Updates an existing user's information.

  • Method: PUT
  • URL: /users/123
  • Query Parameters: None
  • Request Body:
    • {
  "name": "Your Name"
}
  • Response:
    • {
  "id": "123",
  "name": "Your Name"
}
DELETE /users/:id

Deletes a user by their ID.

  • Method: DELETE
  • URL: /users/123
  • Query Parameters: None
  • Request Body: None
  • Response:
    • User with ID 123 was deleted successfully.
